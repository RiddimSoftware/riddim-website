# Riddim Website

Static website source for `riddimsoftware.com`.

## Local Workflow

Install dependencies:

```bash
npm install --package-lock=false
```

Run the local 11ty server:

```bash
npm run dev
```

Build and validate generated output:

```bash
npm run validate
```
